# Project 5 : PHP blog creation without framework.
### Path : Symfony application developer
### Openclassrooms

###### How to install ?
    git clone https://github.com/ThomasG51/OC_DAS_P5.git

    cd OC_DAS_P5

    composer install

###### Components used:
* twig
* symfony/http-foundation
* symfony/dotenv
* symfony/yaml
* symfony/var-dumper (dev)
* symfony/error-handler (dev)
* beberlei/assert
* psr/container
